## Archivos de Inventarios

### Ruta para sacar Inventario

1. Microsoft Visual Studio Deter proceso luego run
2. Ir a escritorio a la carpeta "Archivos de Inventarios"
3. copiar el archivo de `\LOB\Base5`

### Ruta para precios

1. Ejecutar la query `Precios.sql`
2. SQL copiar con headers
3. Cambiar formato a A,D,E a texto y volver a pegar
4. Pegar y guardar como `precios.csv`
   > ⚠️ Asegurarse de respetar el formato nuevo antes de guardar
